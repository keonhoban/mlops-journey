Helm & Ingress 활용 MLflow 서버 외부 PostgreSQL + S3 연동. 모델 등록, 추적, 아티팩트 관리 Airflow 서버 DAG GitSync + S3 연동 + PostgreSQL 외부 DB 사용 FastAPI 서버 MLflow 모델 호출용 예측 API. Ingress로 외부 노출 NFS 서버 Airflow 로그 저장소로 사용.
