### 🧩 `datetime.date`

| 기능 | 설명 |
| --- | --- |
| 날짜 객체 생성 | `datetime.date(연, 월, 일)` |
| 날짜 차이 계산 | `day2 - day1 → timedelta` |
| 요일 확인 | `.weekday()`, `.isoweekday()` 사용 |

```python
import datetime
d1 = datetime.date(2021, 12, 14)
d2 = datetime.date(2023, 4, 5)
print((d2 - d1).days)  # 날짜 차이

print(d1.weekday())     # 0=월, 6=일
print(d1.isoweekday())  # 1=월, 7=일

```

### 🧩 `time` 모듈

| 함수 | 설명 |
| --- | --- |
| `time()` | UTC 기준 초 단위 시간 |
| `localtime()` | 현재 시각을 구조체로 반환 |
| `asctime()` / `ctime()` | 사람이 읽기 좋은 형태로 변환 |
| `strftime()` | 포맷 코드로 날짜 표현 |
| `sleep()` | 코드 실행 일시 정지 |

```python
import time
print(time.ctime())  # 현재 시간 출력
print(time.strftime('%Y-%m-%d %H:%M:%S'))  # 포맷 지정

for i in range(3):
    print(i)
    time.sleep(1)

```

### 🧩 `itertools`


### 🧩 `sympy`

```python
import sympy
from fractions import Fraction

x = sympy.symbols("x")
eq = sympy.Eq(x * Fraction(2, 5), 1760)
sol = sympy.solve(eq)
print(sol[0])  # 4400

# 연립방정식
x, y = sympy.symbols("x y")
eq1 = sympy.Eq(x + y, 10)
eq2 = sympy.Eq(x - y, 4)
print(sympy.solve([eq1, eq2]))  # {x: 7, y: 3}

```
