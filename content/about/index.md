# 👋 안녕하세요!

저는 머신러닝 플랫폼을 직접 설계·구축·운영해 온 MLOps 엔지니어입니다.

2년 3개월간 DevOps 엔지니어로 근무하며 

대규모 시스템 운영, 인프라 안정성, 자동화의 기본기를 쌓았고,

현재는 ML Platform 관점에서 실무 수준의 MLOps 프로젝트를 구축하고 있습니다.

단순히 모델을 배포하는 MLOps가 아니라,
**Data → Feature → Training → Registry → Inference → Monitoring**
전 과정을 하나의 플랫폼으로 연결하고,
운영 중 발생하는 문제를 **구조적으로 해결할 수 있는 엔지니어**를 목표로 하고 있습니다.

---

## 🛠️ 주요 기술 및 실무 경험

### 🔧 MLOps & ML Platform
- **MLflow**: 실험 관리 및 Model Registry 운영
- **Airflow**: 데이터 파이프라인 및 모델 학습 자동화
- **Triton Inference Server**: ONNX 기반 모델 서빙, GitOps 연동
- **Feature Store**
  - Feature Store-lite 직접 설계
  - Feast (S3 Offline Store + Redis Online Store) 구축 및 검증
- **Observability**
  - Prometheus / Grafana 기반 서빙·파이프라인 모니터링
  - Alertmanager를 통한 운영 알람 설계

### ☁️ Infrastructure & DevOps
- Kubernetes 기반 ML 플랫폼 운영
- GitOps (ArgoCD)로 MLOps 컴포넌트 배포 관리
- AWS 인프라 설계 및 자동화 (Terraform)
- CI/CD 및 운영 자동화 경험
