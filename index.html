name: httpd image: httpd ports: - containerPort: 80 --- apiVersion: v1 kind: Service metadata: name: httpd-service spec: selector: app: httpd ports: - port: 80 targetPort: 80 🗺️ [3단계] Ingress 라우팅 설정 🔹 ingress.yaml apiVersion: networking.k8s.io/v1 kind: Ingress metadata: name: example-ingress annotations: nginx.ingress.kubernetes.io/rewrite-target: / spec: rules: - Kong, Ambassador 등) /etc/hosts 설정은 개발 환경 전용 (운영에서는 DNS 서버와 연동) 🔧 MLOps 실전 연결 상황 Ingress 활용 방식 실험 추적 툴 MLflow /mlflow 경로로 노출 노트북 환경 JupyterHub /jupyter 경로로 접속 LLM 서빙 엔드포인트 /v1/chat/completions 등으로 구성 보안 연동 cert-manager + Ingress → TLS 적용
